### Task 3: ECNL RL Staleness Verification Query
**Deliverable:** `Infrastructure/ECNL RL Staleness Verification Query — April 2026.md`

Wrote a purpose-built verification query for ECNL RL (the Source Coverage Priority Matrix's highest-ranked gap: score 9/9). The query:

```sql
SELECT season, COUNT(*) AS total_games,
  MAX(match_date) AS most_recent_game,
  MIN(match_date) AS oldest_game,
  (CURRENT_DATE - MAX(match_date)) AS days_since_most_recent
FROM games
WHERE is_hidden = false
  AND (
    event_tier = 'ecrl'
    OR event_name ILIKE '%ECNL RL%'
    OR event_name ILIKE '%ECNL Regional%'
    OR division ILIKE '%ECNL RL%'
  )
GROUP BY season ORDER BY season DESC;
```

The multi-condition filter is intentional: catches ECNL RL games where `event_tier` classification may not yet be applied, preventing a false PASS from a classification gap.

**PASS threshold:** `days_since_most_recent` ≤ 30 AND current season `total_games` ≥ 3,000.
**YELLOW:** 31–60 days or 1,500–2,999 games → flag in briefing, SENTINEL reviews before Stage 2 decisions.
**FAIL:** > 60 days or < 1,500 games or no current-season row → escalate to SENTINEL immediately, halt Stage 2 planning for ECNL RL-adjacent leagues.
