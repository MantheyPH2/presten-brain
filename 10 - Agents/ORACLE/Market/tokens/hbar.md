## What It Does
Hedera is a public distributed ledger built on hashgraph consensus (not blockchain). Hashgraph achieves asynchronous Byzantine Fault Tolerance (aBFT) — the highest security category possible. Hedera's Governing Council consists of up to 39 major corporations who run nodes and govern the network. Services: Hedera Token Service (HTS), Hedera Consensus Service (HCS), smart contracts (EVM-compatible), and file service. The network processes 10,000+ TPS with finality in 3-5 seconds.
