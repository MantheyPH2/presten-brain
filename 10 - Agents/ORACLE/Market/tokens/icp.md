## What It Does
Internet Computer Protocol (ICP) is a blockchain network created by DFINITY that aims to extend the internet's functionality to host backend software, transforming it into a global compute platform. Unlike Ethereum which runs smart contracts, ICP runs full web applications entirely on-chain — frontend, backend, and data. Key innovations: Chain Key Technology (enables cross-chain interoperability), canisters (smart contracts that serve web content), and the Network Nervous System (on-chain governance). ICP can directly interact with Bitcoin and Ethereum without bridges.

## Why Track This
- **True decentralized cloud computing:** ICP is the most ambitious attempt to replace AWS/Azure/GCP with decentralized infrastructure. If it works, the TAM is $500B+ (cloud computing market)
- **Full-stack on-chain apps:** No other blockchain can host complete web applications (frontend + backend + data) on-chain
- **Bitcoin integration:** ICP can natively hold and transact BTC without bridges — enabling Bitcoin DeFi without wrapping
- **SNS DAOs:** Service Nervous System allows developers to launch DAOs and tokenize applications — a unique governance primitive
- **Massively undervalued if thesis plays out:** $1.4B market cap for a protocol targeting the cloud computing market is either crazy cheap or appropriately priced for a long-shot bet
