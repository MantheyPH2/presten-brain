## What It Does
Filecoin is a decentralized storage network built by Protocol Labs. Miners provide storage capacity and earn FIL tokens. Clients pay FIL to store and retrieve data. The network uses Proof of Replication (proves data is stored) and Proof of Spacetime (proves data is stored over time). Filecoin has over 25 EiB of storage capacity — more than all other decentralized storage networks combined. The Filecoin Virtual Machine (FVM) enables smart contracts on top of the storage layer, expanding use cases to DeFi, data DAOs, and compute-over-data.
