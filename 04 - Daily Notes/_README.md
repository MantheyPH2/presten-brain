# Daily Notes

Daily notes capture what happened each day — wins, work across all three worlds, ideas, and gratitude.

## How it works

- Press `Cmd+N` (or use the Calendar sidebar) to create today's note
- Template auto-fills from `07 - Templates/Daily Note.md`
- Format: `YYYY-MM-DD` (e.g., `2026-04-21`)

## Roll-ups

Daily notes feed into:
- **Weekly Review** — every Sunday, run through [[Weekly Review Template]] to synthesize the week
- **Journal Tracker** — [[Journal Tracker.base|Journal Tracker]] base tracks mood and energy over time
- **Monthly review** — skim the last 4 weekly reviews and adjust quarterly priorities

The daily note is for capture. The weekly review is for thinking.
