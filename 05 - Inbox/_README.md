# Inbox

**Capture everything here first. Weekly review triages into the right world.**

This is the pressure-release valve. When you're in a meeting, have a shower thought, or find an article worth saving — dump it here. Don't waste time deciding where it goes.

## The flow

1. **Capture** — create a note here with whatever's on your mind
2. **Weekly review** — every Sunday, process this folder
3. **Triage** — move each note to the right world (Personal, Tiger Tournaments, Exel Labs, or Reference)
4. **Empty inbox** — goal is zero notes here by end of weekly review

If a note has been here for 3+ weeks and you still haven't sorted it, it's probably not important. Archive it.
